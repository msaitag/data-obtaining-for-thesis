# Python Project to Data Obtaining for Master Thesis
```
This project developed to data obtaining for my Master Thesis and Essay
I used web crawling libraries such as BeautifulSoup and Selenium to obtain data from my sample, Hotel Websites.
First phase of this project is getting all inlinks of every Hotel Websites with sheet name 'llinkDetector.py'.
All inklinks stored into as json file in 'jsonFiles/hotelinlinks.json'.
Second phase of this project is to determine whether the sites contain the required features for each of eMICA(extended Model of Internet Commerce Adoption)'s criteria in the Python sheet which is name 'eMicaItemDetector.py'.
Lastly in the third phase, I collect every result as csv file in 'csvFiles/results.csv'.
I also used Pandas library to workin on datasets and Multiprocessing library to make all process faster.
For more information about my Article DOI number: 'https://doi.org/10.33461/uybisbbd.1344425'.
For more information about my Master Thesis 'https://tez.yok.gov.tr/UlusalTezMerkezi/', thesis no: 793215.
```

# Yüksek Lisans Tezi İçin Python ile Veri Eldesi (Turkish)
```
Bu proje Yüksek Lisans Tezi ve Makale için veri eldesi sağlamak amacıyla geliştirildi.
BeutifulSoup ve Selenium gibi web kazıma kütüphaneleri, örneklem olan Otel Web Sitelerinden veri elde etmek için kullanıldı.
Projenin ilk aşaması her bir otel web sitesinin içerdiği tüm içlinkleri elde etmekdir, 'llinkDetector.py'.
Tüm içlinkler 'jsonFiles/hotelinlinks.json' dosyasında depolanır.
Projenin ikinci aşamasında ise eMICA'nın her bir aşaması için web sitesinin belirlenen kriterleri sağlayıp sağlamadığı tespit edilir, 'eMicaItemDetector.py'.
Projenin son aşamasında ise elde edilen tüm veriler 'csvFiles/results.csv' dosyasında toplanır.
Ayrıca bu projede, veri kümeleri üzerinde çalışmak için Pandas, işlem sürecini kısaltmak için Multiprocessing kütüphanelerini kullandım.
```
